原版的装饰方块上有许多独特的纹理，此模组用这些纹理制作了一系列方块套装。

此模组的配方十分有规律，比如台阶、楼梯等方块的合成方式和原版一致，但是仍有些方块有特殊的配方，推荐安装REI来查阅配方。
